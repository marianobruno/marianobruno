## Hi there 👋

Trabajo en **frameworks de desarrollo asistido por IA** — sistemas y metodologías para llevar
ideas a productos reales de forma más rápida y estructurada.

No solo uso IA para escribir código. La uso para razonar modelos de negocio,
evaluar viabilidad de proyectos antes de construirlos, y diseñar arquitecturas
que sean simples de operar y fáciles de entender.

---

### En qué estoy trabajando

- Frameworks propios para el ciclo completo: **idea → validación → arquitectura → build**
- Evaluación de viabilidad técnica y de negocio con IA como segundo par de ojos
- Infraestructura propia liviana para correr herramientas de IA de forma autónoma
- Portales y sistemas de acceso para ecosistemas de aplicaciones internas

### Principios que guían mi trabajo

- **Sencillo sobre complejo** — si hay una forma más simple, es la correcta
- **Controlable** — lo que construyo tiene que poder operarse sin ser experto
- **Documentado** — cada decisión tiene contexto, cada sistema tiene su manual
- **Evaluar antes de construir** — los planes pasan por revisión de viabilidad antes de una línea de código
